# UI Microcopy (UX Writing)

## Inputs
- Product tone (professional, friendly, strict)
- Domain terms that must be used/avoided

## Output
- Label + helper text suggestions
- Error/confirmation copy
- Empty state copy

## Rules
- Be specific (what happened, what to do)
- Avoid blame
- Keep button text action-based (Verb + Object)
