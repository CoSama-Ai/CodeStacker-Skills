# Forms UX & Validation

## Output
- Form layout guidance (grouping, progressive disclosure)
- Validation strategy (when to validate, message style)
- Error/empty/loading states
- Accessibility checklist for forms

## Best practices
- Inline validation + final submit check
- Errors: specific, actionable, placed near field
- Preserve user input on failure
- Confirm destructive actions
- Support autofill and password managers
