# Navigation & IA

## Inputs
- App modules and user roles
- Primary user journeys
- Current nav structure (if any)

## Output
- Proposed nav hierarchy
- Labeling rules
- Placement rules for settings vs work areas
- Breadcrumb strategy (when needed)

## Guardrails
- Avoid duplicate paths to the same screen
- Keep top-level nav <= 7 items
